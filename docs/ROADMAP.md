# SpeakFit AI Roadmap

## Phase 1: MVP

- Browser voice demo with speech recognition and speech synthesis
- Realtime session endpoint that can issue an OpenAI Realtime session when `OPENAI_API_KEY` is set
- Transcript-based speaking assessment JSON
- Daily mission completion
- Streak and weekly score persistence
- Mobile-first UI using the provided coach and child characters

## Phase 2: Personalized Coach

- Replace demo assessment with model-generated scoring
- Add repeated-error analysis across sessions
- Build review cards and shadowing drills
- Add push notifications and streak freeze
- Add goal-specific tracks for travel, fintech meetings, investor pitch, and conference booth practice

## Phase 3: B2B / Education

- Admin dashboard
- Team-level speaking reports
- Organization-specific scenario generation
- Industry-specific learning packs
- Licensing, cohort analytics, and weekly team ranking
